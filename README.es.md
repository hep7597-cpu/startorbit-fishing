# Fishing Joy - Juego WebGL Oficial de Alto Rendimiento para GGEMU.COM

[🇬🇧 English](./README.md) | [🇨🇳 中文](./README.zh.md) | [🇯🇵 日本語](./README.ja.md) | [🇪🇸 Español](./README.es.md)

Este proyecto es un juego de Pesca (Fishing Joy) de nivel profesional desarrollado específicamente para la plataforma **[GGEMU.COM](https://ggemu.com/es)**. Demuestra cómo construir juegos WebGL de alto rendimiento utilizando **PixiJS v8** e integrar perfectamente el **GGEMU SDK** para funciones sociales y económicas.

**[GGEMU](https://ggemu.com/es)** es una plataforma líder en la nube para juegos retro y de navegador. Redefine la experiencia de juego al traer miles de clásicos a la web moderna.

---

## 🔥 Características Técnicas

Como juego insignia de HTML5 para GGEMU, este proyecto presenta varias tecnologías de vanguardia:

*   **🌊 Shaders de Ondas de Agua Avanzados**: Shaders personalizados de WebGL 2.0 que brindan refracción dinámica e iluminación realistas.
*   **📈 Motor de Probabilidad RTP**: Algoritmo de captura de grado profesional que equilibra la emoción del jugador con la economía de la plataforma (Return to Player).
*   **🎏 Formaciones de Peces Impresionantes**: Planificación sofisticada de rutas vectoriales para comportamientos de bancos de peces y patrones geométricos.
*   **🔊 Audio Profesional de Baja Latencia**: Motor de mezcla multicanal impulsado por **Tone.js**, garantizando una sincronización perfecta.

## 🛠️ Potencia del [GGEMU SDK](https://ggemu.com/es)

La integración con el **GGEMU SDK** transforma este juego en una experiencia comercialmente lista:

### 1. 🪙 Ecosistema Unificado de GCoin
*   **Activos Globales**: El oro ganado en Fishing Joy se sincroniza con la **Cuenta Global de GGEMU (GCoin)**.
*   **Transacciones Seguras**: APIs confiables para la gestión de activos en tiempo real.

### 2. 📸 Captura de Medios en Alta Definición
*   **Capturas de pantalla**: Captura momentos impresionantes directamente desde el búfer de renderizado.
*   **Grabación de vídeo**: Flujo de grabación eficiente para compartir momentos destacados.

### 3. 📡 Servicio Global en Vivo
*   **Streaming Instantáneo**: El SDK registra automáticamente el Canvas y AudioNode, permitiendo que el juego aparezca en el feed de **Live** de la plataforma.
*   **Presencia Social**: Actualizaciones de estado en tiempo real.
