# 欢乐捕鱼 (Fishing Joy) - GGEMU.COM 公式 WebGL ゲーム

[🇬🇧 English](./README.md) | [🇨🇳 中文](./README.zh.md) | [🇯🇵 日本語](./README.ja.md) | [🇪🇸 Español](./README.es.md)

本プロジェクトは、**[GGEMU.COM](https://ggemu.com/ja)** プラットフォーム向けに開発されたプロフェッショナル仕様の「捕魚ゲーム」です。**PixiJS v8** を活用した高性能な WebGL ゲームの構築方法と、**GGEMU SDK** によるソーシャル・経済機能の統合方法を実証しています。

## 🔥 技術的特徴

GGEMU 向けのフラッグシップ HTML5 ゲームとして、以下の先端技術を採用しています：

*   **🌊 高性能水波シェーダー**: リアルな水の屈折と照明効果を実現するカスタム WebGL シェーダー。
*   **📈 最適化された RTP 確率エンジン**: RTP（還元率）フィードバックを利用し、プレイヤーの爽快感とバランスを両立。
*   **🎏 壮大な魚群陣列**: ベクターパスによる複雑な魚群の行動と幾何学的な陣列を再現。
*   **🔊 低遅延プロオーディオ**: **Tone.js** によるマルチチャンネルミキシングにより、完璧な音画同期を実現。

## 🛠️ [GGEMU SDK](https://ggemu.com/ja) の力

**GGEMU SDK** との連携により、本ゲームは単なるデモを超えた商業品質を実現しています：

### 1. 🪙 統合 GCoin システム
*   **グローバル残高**: 獲得したゴールドは **GGEMU グローバルアカウント (GCoin)** にリアルタイム同期されます。
*   **安全なトランザクション**: `useBagCoins` や `addBagCoins` インターフェースによる確実な資産管理。

### 2. 📸 高画質メディアキャプチャ
*   **スクリーンショット**: レンダリングバッファから直接、高画質な瞬間をキャプチャ。
*   **録画機能**: 効率的な録画パイプラインにより、ハイライトをコミュニティで共有。

### 3. 📡 グローバルライブサービス
*   **ライブストリーミング**: SDK が Canvas と AudioNode を自動登録し、プラットフォームの **Live** チャンネルにゲーム画面を転送します。
*   **ソーシャルプレゼンス**: プレイ状況をリアルタイムに更新し、友達にアピール。
