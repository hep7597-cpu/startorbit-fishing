## 🔥 项目技术特性

本项目作为 GGEMU 平台的原生 HTML5 精品游戏，展现了多项尖端技术：

*   **🌊 高阶水波纹着色器 (Water Filters)**：
    基于 WebGL 2.0 深度定制的流体着色器，实现了极致自然的动态水折射与光影叠加，赋予画面极其通透的视觉质感。

*   **📈 精心调校的 RTP 概率引擎**：
    内置一套“静心调制”的商业级捕获算法。利用 RTP 动态反馈逻辑均衡玩家的投入与产出，完美兼顾了游戏的挑战性与爽快感。

*   **🎏 极富视觉冲力力的鱼群阵列**：
    支持复杂的矢量路径规划，能够呈现大规模且错落有致的鱼群阵列巡航效果。

*   **🔊 专业级低延迟音效系统**：
    基于 **Tone.js** 打造的多通道混音引擎。针对打击、网散、背景波纹声进行了精细化处理，确保在激烈对战中音画完美同步。
